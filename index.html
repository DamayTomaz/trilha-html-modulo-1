O que são tags html?

"tags" são elementos usados para marcar e estruturar o conteúdo em uma página da web. Elas fornecem instruções para o navegador da web sobre como o conteúdo deve ser exibido e interpretado.
As tags HTML são cercadas por colchetes angulares e geralmente vêm em pares, consistindo em uma "tag de abertura" e uma "tag de fechamento"

h1-h6

As teclas H em HTML se referem às tags de cabeçalho de nível (heading tags) de H1 a H6.

Essas tags são usadas para criar cabeçalhos ou títulos em uma página da web e são estruturadas hierarquicamente, com H1 sendo o mais importante e H6 o menos importante.

P

Parágrafo:

A tag

é usada para criar parágrafos de texto em uma página HTML, organizando o conteúdo em blocos legíveis e lógicos.

Mark

Marcação:

A tag mark é usada para destacar ou marcar parte do texto em destaque, geralmente com um fundo amarelo por padrão.

Exemplo:

Estudar na Dio é muito bom!

Small

Texto pequeno

A tag é usada para indicar que o texto dentro dela deve ser exibido em um tamanho de fonte menor do que o texto circundante.

Exemplo:

Estudar na dio é muito bom!

I

Texto em Itálico

A tag i é usada para aplicar estilo de itálico ao texto

Exemplo:

Estudar na Dio é muito bom!

U

Texto Sublinhado

A tag u é usada para aplicar sublinhado ao texto

Exemplo:

Estudar na Dio é muito bom!]

Strong

Texto Importante

A tag strong é usada para enfatizar o texto, indicndo que ele é de importância significativa.

Exemplo:

Estudar na Dio é muito bom!

OL

Lista Ordenada:

A tag ol é usada para criar listas ordenadas, onde os itens são numerados sequencialmente

Exemplo:

  1. Criatividade
  2. Honestidade
  3. Verdade

UL

Listas Não ordenadas

A tag ul é usada para criar listas não ordenadas, onde os itens são apresentados como marcadores

Exemplo:

  • Criatividade
  • Honestidade
  • Verdade

LI

Item de Lista

a tag li é usada dentro de listas (<"ol"> ou <"ul">) para definir itens individuais da lista

Exemplo:

  • Lealdade
  • Respeito
  • Diversidade

    Texto sobrecrito

    A tag sub é usada para colocar texto em uma posição inferior e menor do que o texto circundante, frequentemente usado para indicar subscritos.

    Estudar na Dio é muito bom!

    Bloco de citação

    A tag blockquote é usada para criar blocos de texto que são citados de outra fonte.

    Exemplo:

    FONT

    Escrita

    A tag font é usada para definir a formataçãp de fonte, como tamanho, cor e familia de fonte.

    Exemplo:

    Estudar na Dio é muito bom

    DEL

    Texto Deletado

    A tag del é usada para marcar texto que foi removido ou deletado de uma versão anterior do conteúdo

    Exemplo:

    Estudar na Dio é muito bom!

    ABBR

    Abreviação

    A tag abbr é usada para marcar abreviações em um texto e, opcionalmente, fornecer uma descrição completa do que a abreviação significa.

    Exemplo:

    A Dio é muito boa.
